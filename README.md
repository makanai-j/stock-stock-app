## はじめる

### ビルド

クローン

```bash
git clone https://github.com/makanai-j/stock-stock-app.git
```

依存関係インストール

```bash
npm install
```

ビルドスタート

```bash
npm run start
```

### 前提条件

node.js

> 開発時バージョン
> v20.12.2

[node.js install](https://nodejs.org/en/download/package-manager)
